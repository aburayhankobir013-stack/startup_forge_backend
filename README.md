# STARTUP FORGE BACKEND
# 🚀 StartupForge - Server

StartupForge Server provides REST APIs for authentication, startup management, opportunity management, applications, payments, and admin dashboard.


## ✨ Features

- Better Auth Authentication
- JWT Authorization
- Role Based Access Control
- MongoDB CRUD Operations
- Startup CRUD
- Opportunity CRUD
- Application CRUD
- Stripe Payment Integration
- Admin Dashboard APIs
- Founder Dashboard APIs
- Collaborator Dashboard APIs
- Search API
- Filter API
- Pagination API
- Secure Environment Variables
- Error Handling

---

## 🛠️ Technologies Used

- Node.js
- Express.js
- MongoDB
- Better Auth
- JWT
- Stripe
- CORS
- Dotenv
